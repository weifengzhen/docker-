# docker的容器命令操作

### 需要关闭虚拟机防火墙

![1593232690917](assets/1593232690917.png)

![1593232607284](assets/1593232607284.png)



1docker运行mysql

**docker run --name aaa -d mysql：5.5**

aaa：自定义名字

-d：指在后台运行

mysql：运行的软件名字

5.5：版本号

**运行tomecat**

docker run -d -p 8888:8080 tomcat

-p:指定端口号

8888：虚拟机的端口

8080：tomcat的端口
